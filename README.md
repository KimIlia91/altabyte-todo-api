# Todo List API

## Локальная разработка

```bash
# Установка зависимостей
make install

# Запуск API
make run
```

## Миграции

```bash
cd backend
alembic upgrade head
```

## Команды

- `make build` - собрать Docker образы
- `make up` - запустить контейнеры
- `make down` - остановить контейнеры
- `make logs` - показать логи
